# 成本对比分析技能

## 0. 关键概念定义（重要）

### 0.1 建面单方的定义

**建面单方 = 成本科目总金额 / 总建筑面积**

- **总建筑面积**：项目的总建筑面积（如树村地块129,947.34㎡，摩天轮地块55,399.374㎡）
- **注意区分**：
  - **建面单方**：成本科目总金额 / 总建筑面积（用于汇总对比）
  - **相对建筑面积单方**：成本科目总金额 / 相对建筑面积（用于各业态内部对比）
- **重要说明**：各业态的"相对建筑面积单方"不能直接相加得到合计行的"建面单方"，因为分母不同

### 0.2 业态对应关系

| 项目类型 | 业态名称 | 对应关系 |
|---------|---------|---------|
| 树村地块 | 叠拼 | 对应摩天轮叠墅 |
| 树村地块 | 洋房 | 对应摩天轮小高层 |
| 摩天轮地块 | 叠墅 | 对应树村叠拼 |
| 摩天轮地块 | 小高层 | 对应树村洋房 |

### 0.3 含量影响和价格影响计算公式

```
含量影响 = (含量1 - 含量2) × 平均单价
价格影响 = (单价1 - 单价2) × 平均含量
平均单价 = (单价1 + 单价2) / 2
平均含量 = (含量1 + 含量2) / 2

验证：含量影响 + 价格影响 ≈ 单方差异（允许微小误差）
```

### 0.4 数据合并处理要点（⚠️ 严格执行，避免数据翻倍错误）

**【核心原则】合并科目的单方必须从05表的汇总行直接读取，严禁从子项行相加后再次计算！**

05表中存在多级结构：
- **三级科目汇总行**（如`03.04.01.01钢筋工程`）：其单方 = 所有四级子项单方之和
- **四级科目明细行**（如`03.04.01.01.01钢筋主材`、`03.04.01.01.02钢筋制安`）：各自有独立的含量和单价

**数据提取规则（必须严格遵守）：**

#### 钢筋工程
05表中钢筋工程包含"钢筋主材"和"钢筋制安"两个四级子项：
- **单方数据来源**：必须从三级科目汇总行`03.04.01.01钢筋工程`的**J列（相对建筑面积单方）**直接读取，**严禁**从子项行分别读取后相加再乘以任何系数
- **含量数据来源**：从四级子项`03.04.01.01.01钢筋主材`的**G列（含量/系数）**读取
- **单价数据来源**：合并单价 = 汇总行单方 / 含量（即 主材单价 + 制安单价）
- **典型错误**：从子项行分别读取主材单方164.94和制安单方136.03，相加得300.97，再错误地乘以2得到601.94（翻倍错误）
- **正确做法**：直接读取汇总行J列=300.97，含量=49.83，合并单价=300.97/49.83=6.04
- **影响分解公式**：
  - 合并单价1 = 单方1 / 含量1，合并单价2 = 单方2 / 含量2
  - 含量影响 = (含量1 - 含量2) × (合并单价1 + 合并单价2) / 2
  - 价格影响 = (合并单价1 - 合并单价2) × (含量1 + 含量2) / 2

#### 混凝土工程
05表中混凝土工程包含"混凝土主材"、"混凝土浇筑"和可能的"预制构件采购/安装"等子项：
- **单方数据来源**：必须从三级科目汇总行`03.04.01.03混凝土工程`的**J列**直接读取
- **含量数据来源**：从四级子项`03.04.01.03.01混凝土主材`的**G列**读取
- **单价数据来源**：合并单价 = 汇总行单方 / 含量（注意：因包含预制构件等子项，合并单价≠主材单价+浇筑单价）
- **影响分解公式**：同钢筋工程，使用合并单价计算
- **特别注意**：混凝土工程可能包含预制构件等非含量×单价结构的子项，此时合并单价会显著高于主材+浇筑单价之和

#### 粗装修工程
- **单方数据来源**：必须从三级科目汇总行`03.04.01.05粗装修工程`的**J列**直接读取
- **含量和单价**：粗装修无标准的含量×单价结构，C/D/F/G列填1和单方值即可
- **严禁翻倍**：如源数据粗装修单方=400，对比表中必须填400，不能填800

#### 开办费
- **单方数据来源**：必须从三级科目汇总行`03.04.01.08开办费`的**J列**直接读取
- **含量和单价**：开办费无标准的含量×单价结构，C/D/F/G列填1和单方值即可
- **注意**：开办费的含量字段（如27.44）是工期月数等含义，不是标准含量，不能用于含量×单价验证

### 0.5 单方数据提取防错规则（⚠️ 关键）

**【强制要求】生成对比表后，必须逐项验证以下规则，任何一条不通过都必须修正：**

#### 规则1：合并科目单方验证
对于钢筋、混凝土、粗装修、开办费等合并科目：
- 对比表中的单方值 必须等于 05表对应业态汇总行J列的值
- 验证方法：`对比表单方 / 源数据汇总行J列 ≈ 1.0`（误差<0.01）
- 如果比值≈2.0，说明数据被错误翻倍，必须修正

#### 规则2：合并科目含量×单价验证
- **钢筋、混凝土**：合并单价 = 单方 / 含量，验证 `含量 × 合并单价 ≈ 单方`
- **模板、砌筑**：验证 `含量 × 单价 = 单方`
- **粗装修、开办费**：不适用此验证（数据结构不同）

#### 规则3：影响计算验证
- 对于所有有含量和单价的科目：`含量影响 + 价格影响 ≈ 单方差异`（误差<0.1）

#### 规则4：差异说明完整性验证
- O列（固定差异说明）：必须包含所有固定科目的差异，建筑工程子项中钢筋、模板、混凝土、砌筑必须标注【含量影响XX元/m²，价格影响XX元/m²】
- P列（弹性差异说明）：必须包含所有弹性科目的差异，入户门、外立面门窗、安防、电梯必须标注【含量影响XX元/m²，配置影响XX元/m²】

### 1. 固定差异说明详细分析要求

**核心流程（必须严格执行）：**
1. 从05表S列识别该业态下所有属性为"固定"的二级科目
2. 计算每个固定科目的单方差异（项目1单方 - 项目2单方）
3. **只列出有差异的科目**（单方差异不为0的科目），差异为0的可以不列
4. **最终校验：所有列出的固定科目差异之和必须等于该业态固定差异总数**
5. 如果校验不通过，必须检查是否有遗漏的科目

**格式要求：**
- 第一行：`相对建筑面积口径固定成本单方+X,XXX.XX元/m²；其中：`（正数用+号，负数用-号，数字千分位分隔保留2位小数）
- 建筑工程作为第一个科目，需展开子项（钢筋、模板、混凝土、砌筑、粗装修、保温及其他、开办费）

- **粗装修和开办费不标注含量/价格影响**（因其无标准含量×单价结构）
- 其余固定科目按顺序列出，格式为`N、XXX工程+/-XXX.XX元/m²`
- 同样名字的科目实际是父级科目和子级科目的关系，不用重复列入

## 数据交叉验证规则（⚠️ 生成后必须执行）

**【强制要求】每次生成成本对比分析表后，必须执行以下交叉验证，任何一条不通过都必须修正后重新生成：**

### 验证1：单方数据来源验证
- 对比表中每个合并科目（钢筋、混凝土、粗装修、开办费）的单方值，必须回溯到05表对应业态的汇总行J列进行比对
- 验证方法：`对比表单方 / 源数据汇总行J列 ≈ 1.0`（误差<0.01）
- 如果比值≈2.0，说明数据被错误翻倍，必须从源数据重新提取

### 验证2：含量×单价一致性验证
- **钢筋、混凝土**：合并单价 = 单方 / 含量，验证 `含量 × 合并单价 ≈ 单方`
- **模板、砌筑**：验证 `含量 × 单价 = 单方`
- **粗装修、开办费**：跳过（数据结构不同，无标准含量×单价关系）

### 验证3：影响分解验证
- 对于所有有含量和单价的科目（钢筋、模板、混凝土、砌筑）：`含量影响 + 价格影响 ≈ 单方差异`（误差<0.1）
- 如果不通过，检查I列和J列公式是否正确

### 验证4：差异说明完整性验证
- **O列（固定差异说明）**：
  - 必须包含所有固定科目的差异（不只是建筑工程子项）
  - 建筑工程子项中，钢筋、模板、混凝土、砌筑必须标注【含量影响XX元/m²，价格影响XX元/m²】
  - 粗装修和开办费不标注含量/价格影响
  - 所有固定科目差异之和 ≈ 第一行的固定成本单方总差异
- **P列（弹性差异说明）**：
  - 必须包含所有弹性科目的差异
  - 入户门、外立面门窗、安防、电梯必须标注含量影响和配置影响
  - 所有弹性科目差异之和 ≈ 第一行的弹性成本单方总差异

### 验证5：公式完整性验证
- 固定差异明细表：所有业态的H/I/J列都必须有公式（粗装修和开办费除外，只有H列公式）
- 弹性差异明细表：所有业态的B/E/H列都必须有公式
- 选中任意计算单元格，公式栏应显示公式而非数值

### 验证6：源数据一致性验证
- 对比表中的总价数据应与04表一致
- 对比表中的单方数据应与05表汇总行一致
- 对比表中的含量和单价数据应与05表子项行一致
